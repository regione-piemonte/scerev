# Componente di Prodotto

SCEREVBE

## Versione

1.1.1

## Descrizione

API REST per la scelta e la revoca del medico a fruizione del frontend rivolto al cittadino.

## Prerequisiti di sistema

Java:
AdoptOpenJDK 11.0.4

ANT:
Ant version 1.8.4

Server Web:
Apache 2.4.53

Application Server:
Wildfly 17.0.0

Tipo di database:
postgres v9.6.19

Contiene riferimenti a SCEREVJPA, SCEREVUTIL, SCEREVSERVIZIESTERNI.
