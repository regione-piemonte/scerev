# Componente di Prodotto

SCEREVJPA

## Versione

1.1.1

## Descrizione

Componente di gestione della persistenza dei dati, utilizzata da SCEREVBOWEB e SCEREVBE.

## Prerequisiti di sistema

Java:
AdoptOpenJDK 11.0.4

ANT:
Ant version 1.8.4

Server Web:
Apache 2.4.53

Application Server:
Wildfly 17.0.0

Tipo di database:
postgres v9.6.19
