# Componente di Prodotto

SCEREVBOWEB

## Versione

1.1.1

## Descrizione

WebApp di backoffice ad uso degli operatori sanitari, con funzioni complementari alla gestione della scelta e della revoca del cambio del medico.

## Prerequisiti di sistema

Java:
AdoptOpenJDK 11.0.4

ANT:
Ant version 1.8.4

Server Web:
Apache 2.4.53

Application Server:
Wildfly 17.0.0

Tipo di database:
postgres v9.6.19

Contiene riferimenti a SCEREVBOWCL, SCEREVJPA, SCEREVUTIL, SCEREVSERVIZIESTERNI.
